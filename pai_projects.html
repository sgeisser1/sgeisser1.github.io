Course Overview

This Probabilistic Artificial Intelligence course at ETH Zurich provided comprehensive training in Bayesian machine learning methods and uncertainty quantification. The course emphasized the importance of probabilistic thinking in AI applications, covering fundamental concepts from Gaussian Processes to modern Bayesian deep learning techniques.

Each project addressed real-world challenges where uncertainty quantification is crucial: environmental monitoring, medical imaging, drug discovery, and control systems. The coursework emphasized both theoretical understanding and practical implementation of state-of-the-art probabilistic AI methods.

Project 1: Pollution Concentration Prediction with Gaussian Processes

Implemented Gaussian Process Regression for predicting PM2.5 pollution concentrations with spatial uncertainty quantification. This project addressed the critical challenge of environmental monitoring with limited sensor data and incorporated city-area constraints for realistic policy applications.

  • Spatial Gaussian Processes: Developed GP models for 2D spatial prediction using Matérn and RBF kernels with optimized hyperparameters
  • Constrained Optimization: Implemented city-area constraints where underprediction in urban zones incurs higher costs, reflecting real-world policy implications
  • Kernel Design: Experimented with multiple kernel types (Matérn ν=2.5, RBF, White noise) and automatic hyperparameter optimization using multiple restarts
  • Uncertainty-Aware Predictions: Leveraged GP posterior variance to make risk-aware predictions, adding standard deviation factors in high-risk city areas
  • Cost Function Optimization: Designed asymmetric cost functions penalizing underprediction (cost=3) vs. overprediction (cost=1) in city areas
  • 3D Visualization: Created comprehensive 3D surface plots combining training data, predictions, and uncertainty estimates for model interpretation

Project 2: Bayesian Neural Networks for Satellite Image Classification

Implemented SWAG (SWA-Gaussian) for uncertainty-aware land-use classification from satellite imagery. This project addressed the critical need for calibrated uncertainty estimates in Earth observation applications where ambiguous samples require careful handling.

  • SWAG Implementation: Developed both SWAG-Diagonal and full SWAG for approximate Bayesian inference in neural networks, building on Stochastic Weight Averaging
  • Satellite Image Processing: Processed 60×60 RGB satellite images for 6-class land usage classification (forests, water bodies, urban areas, agriculture, etc.)
  • Uncertainty Calibration: Implemented Expected Calibration Error (ECE) measurement with 20 bins to ensure model confidence matches actual accuracy
  • Ambiguous Sample Detection: Developed methods to identify and handle ambiguous samples (mixed land usage, seasonal variations, cloud cover) using confidence thresholds
  • Cost-Sensitive Learning: Optimized prediction strategies balancing "don't know" responses (cost=1) against incorrect classification (cost=3)
  • Model Averaging: Implemented Bayesian model averaging over multiple posterior samples for robust predictions with quantified uncertainty

Project 3: Constrained Bayesian Optimization for Drug Discovery

Developed safe Bayesian optimization algorithms for drug candidate development, optimizing bioavailability (logP) while respecting synthetic accessibility constraints. This project addressed real challenges in pharmaceutical research where safety constraints are paramount.

  • Safe Bayesian Optimization: Implemented constrained BO ensuring synthetic accessibility (SA < 4.0) while maximizing bioavailability (logP)
  • Multi-Objective Gaussian Processes: Developed separate GP models for objective function (Matérn kernel) and constraint function (additive Linear + Matérn kernels)
  • Acquisition Function Design: Implemented acquisition functions respecting feasibility constraints and balancing exploration-exploitation in safe regions
  • Lagrangian Relaxation: Applied constrained optimization techniques using penalty methods: max f(x) - λ max(v(x), 0)
  • Hyperparameter Optimization: Automated kernel parameter tuning for both objective and constraint functions using maximum likelihood estimation
  • Drug Design Applications: Demonstrated practical relevance for pharmaceutical research with realistic molecular property constraints

Project 4: Off-Policy Reinforcement Learning for Control

Implemented off-policy RL algorithms for inverted pendulum control, learning swing-up policies from simulation data. This project demonstrated modern RL techniques for continuous control problems with safety considerations.

  • Off-Policy Learning: Developed algorithms that learn optimal policies from previously collected data without online interaction during training
  • Pendulum Control: Solved the classic inverted pendulum swing-up problem (from π to 0 angle) requiring precise control and timing
  • Continuous Action Spaces: Handled continuous torque control rather than discrete actions, requiring advanced policy representation
  • Sample Efficiency: Optimized learning from limited simulation data, crucial for real-world applications where data collection is expensive
  • Policy Evaluation: Implemented robust policy evaluation methods for assessing control performance in simulation environments
  • Control Theory Integration: Combined RL with classical control theory principles for stable and effective pendulum manipulation
