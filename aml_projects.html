Course Overview

This Advanced Machine Learning course at ETH Zurich provided hands-on experience with cutting-edge ML techniques across diverse domains. The course emphasized practical implementation of state-of-the-art algorithms, addressing real-world challenges in medical imaging, biomedical signal processing, and regression analysis. Each project required not only technical implementation but also careful consideration of evaluation metrics, data preprocessing, and model interpretability.

The projects spanned multiple domains from healthcare applications to computer vision, providing comprehensive exposure to modern machine learning methodologies and their practical applications in research and industry settings.

Project 1: Brain Age Prediction

Developed machine learning models to predict brain age from neuroimaging data, addressing one of the key challenges in neuroscience and aging research. This project involved working with high-dimensional medical imaging data and implementing regression techniques for age estimation.

  • Neuroimaging Analysis: Processed and analyzed brain imaging data to extract meaningful features correlating with biological age
  • Regression Modeling: Implemented various regression algorithms to predict chronological age from brain structure features
  • Feature Selection: Applied dimensionality reduction techniques to handle high-dimensional neuroimaging data effectively
  • Cross-Validation: Employed robust validation strategies to ensure model generalization and prevent overfitting
  • Medical Applications: Explored potential clinical applications for brain age prediction in neurodegenerative disease research

Project 2: ECG Heart Rhythm Classification

Implemented a comprehensive machine learning pipeline for classifying heart rhythms from raw ECG signals, addressing challenges in biomedical signal processing and automated cardiac diagnosis. The project involved sophisticated preprocessing and feature extraction techniques for time series data.

  • Signal Preprocessing: Developed robust preprocessing pipelines including Butterworth filtering (0.4-4 Hz) to remove noise from respiration and power supply interference
  • Data Segmentation: Implemented windowing strategies with 60-80% overlap to segment variable-length ECG recordings into fixed-size analysis windows
  • Feature Engineering: Extracted both supervised features (statistical measures, HeartPy library features) and unsupervised features using CNN-based approaches on spectrograms and STFT representations
  • Multi-Class Classification: Trained and compared multiple algorithms including SVM, Gradient Boosting (XGBoost, LightGBM, CatBoost), Random Forest, and K-Nearest Neighbors for 4-class heart rhythm classification
  • Model Ensemble: Combined different classification models to improve overall performance and robustness using ensemble techniques
  • Evaluation Metrics: Optimized models using micro-averaged F1-score, particularly important for imbalanced medical datasets

Project 3: Mitral Valve Segmentation

Developed computer vision algorithms for automated mitral valve segmentation in echocardiography videos, contributing to cardiovascular disease diagnosis and treatment planning. This project combined medical domain knowledge with advanced deep learning techniques.

  • Medical Image Analysis: Processed echocardiography videos to identify and segment mitral valve structures across multiple frames
  • Video Processing: Handled temporal consistency in medical video data with 65 training videos containing labeled frames for supervision
  • Semantic Segmentation: Implemented pixel-level classification to precisely delineate mitral valve boundaries in ultrasound imagery
  • Multi-Frame Analysis: Developed techniques to process multiple frames per video (typically 3 labeled frames) for robust segmentation
  • Expert vs Amateur Data: Handled mixed-quality annotations from both expert cardiologists and amateur annotators to improve model robustness
  • Evaluation Methodology: Achieved performance measurement using median Jaccard Index (IoU) across all test videos, standard metric for medical segmentation tasks
