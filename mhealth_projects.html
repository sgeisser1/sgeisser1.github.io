Course Overview

This cutting-edge mobile health course focused on developing algorithms for wearable technology applications, particularly in the realm of health monitoring and activity recognition. Through two comprehensive exercises, I implemented advanced signal processing and machine learning techniques for real-world mobile health scenarios that directly parallel commercial smartwatch and fitness tracker technologies.

The course emphasized practical implementation of algorithms that power modern wearable devices, covering the complete pipeline from raw sensor data acquisition to actionable health insights. Each project addressed real challenges faced by companies like Apple, Samsung, Polar, and Garmin in their wearable product development.

Exercise 1: Advanced Heart Rate Estimation from PPG Signals

Developed robust heart rate monitoring algorithms using photoplethysmography (PPG) signals from wearable devices, addressing the core challenge of accurate HR estimation in noisy, real-world conditions.

  • PPG Signal Processing: Implemented advanced filtering and preprocessing techniques for raw PPG signals from forehead and wrist sensors, handling 19-bit unsigned integer data at 128 Hz sampling rate
  • Noise Robust Algorithms: Developed algorithms resilient to motion artifacts and poor signal quality typical in wearable device usage scenarios
  • Multi-Modal Enhancement: Integrated accelerometer data to improve heart rate estimation accuracy through motion compensation and signal quality assessment
  • Peak Detection & Analysis: Implemented sophisticated peak detection algorithms using SciPy's find_peaks() with optimized parameters for physiological heart rate ranges (40-180 BPM)
  • Temporal Windowing: Designed sliding window approaches for 30-second heart rate calculations, balancing temporal resolution with measurement accuracy
  • Frequency Domain Analysis: Applied FFT-based spectral analysis for heart rate extraction in challenging signal conditions

Exercise 2: Comprehensive Wearable Activity Monitoring System

Built a complete activity monitoring system capable of step counting, device location detection, activity recognition, and path classification using real-world crowd-sourced data from 676 recordings.

  • Multi-Task Learning Framework: Developed unified system addressing four distinct but related tasks: step counting, smartwatch location classification, activity recognition, and path identification
  • Robust Step Counting: Implemented accurate step detection algorithms for natural walking, running, and stair climbing activities, handling edge cases and irregular movement patterns
  • Device Location Classification: Built machine learning models to classify smartwatch placement (left wrist, belt/waistband, right ankle) using accelerometer and gyroscope signatures
  • Activity Recognition: Designed classifiers to detect standing, walking, running, and cycling activities with minimum 60-second duration requirements
  • Geographic Path Classification: Implemented algorithms to identify specific routes between Zurich main station and ETH using motion and elevation patterns
  • Real-World Data Processing: Handled complex, noisy sensor data from LilyGo smartwatches and smartphones with varying sampling rates and missing data

Advanced Signal Processing Techniques

Throughout both exercises, I implemented sophisticated signal processing methods essential for mobile health applications:

  • Adaptive Filtering: Designed adaptive filters for removing motion artifacts from PPG signals while preserving cardiac information
  • Feature Engineering: Extracted time-domain and frequency-domain features from accelerometer, gyroscope, and magnetometer data for activity classification
  • Multi-Sensor Fusion: Developed algorithms combining data from multiple sensors (accelerometer, gyroscope, magnetometer) for improved accuracy
  • Real-Time Constraints: Optimized algorithms to meet strict 10-second processing requirements for real-time wearable device applications
  • Robust Statistics: Applied robust statistical methods to handle outliers and noise typical in mobile sensor data

Machine Learning and Classification

Implemented advanced machine learning techniques for complex multi-class and multi-task problems:

  • Feature Selection: Applied statistical and information-theoretic methods to identify optimal feature sets for each classification task
  • Weak Label Learning: Handled challenging weakly-labeled activity data where precise temporal boundaries were unknown
  • Multi-Class Classification: Developed classifiers for device location (3 classes), activities (4 classes), and paths (5 classes)
  • Temporal Modeling: Incorporated temporal dependencies and sequence information for improved activity recognition
  • Cross-Validation: Implemented robust evaluation strategies appropriate for time-series and person-specific data
